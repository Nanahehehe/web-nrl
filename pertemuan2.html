Artikel: Array dan String dalam Pemrograman

Array adalah struktur data yang menyimpan sekumpulan nilai dalam satu variabel. Sedangkan String adalah kumpulan karakter yang digunakan untuk merepresentasikan teks dalam pemrograman.

1. Contoh Implementasi Array di Python

        
# Contoh array dalam Python
angka = [10, 20, 30, 40, 50]

# Mengakses elemen array
print(angka[0])  # Output: 10

# Menambahkan elemen ke array
angka.append(60)

# Menghapus elemen dari array
angka.remove(20)

2. Contoh Manipulasi String di Python

        
# Contoh String di Python
teks = "Hello, Nurul!"

# Mengakses karakter pertama
print(teks[0])  # Output: H

# Mengubah huruf menjadi kapital
print(teks.upper())  # Output: HELLO, NURUL!

# Menggabungkan string
nama = "Nurul"
sapaan = "Halo, " + nama
print(sapaan)  # Output: Halo, Nurul
